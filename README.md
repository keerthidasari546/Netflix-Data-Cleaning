# Netflix Movies & TV Shows Data Cleaning

## Project Overview
This project demonstrates data cleaning and preprocessing using the Netflix Movies and TV Shows dataset with Python and Pandas.

## Objective
- Handle missing values
- Remove duplicate records
- Standardize text values
- Convert date formats
- Rename column headers
- Prepare the dataset for analysis.

## Tools Used
- Python
- Pandas
- Google Colab

## Dataset
Netflix Movies and TV Shows Dataset (Kaggle)

## Data Cleaning Steps
1. Loaded the dataset using Pandas.
2. Checked for missing values.
3. Filled missing values in `director`, `cast`, and `country`.
4. Removed rows with missing `date_added`.
5. Removed duplicate records.
6. Renamed column names to lowercase with underscores.
7. Standardized text values.
8. Converted `date_added` to a consistent date format.
9. Saved the cleaned dataset as `cleaned_netflix_dataset.csv`.

## Files Included
- `netflix_titles.csv` - Original dataset
- `cleaned_netflix_dataset.csv` - Cleaned dataset
